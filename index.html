Planes

Opciones segun el momento de tu marca.

No todas las marcas necesitan lo mismo. Algunas solo necesitan una buena landing, y otras ya estan listas para una tienda completa con una imagen mas personalizada.

PLAN 01

Presencia profesional / Landing de conversion

Para marcas o profesionales que necesitan estar online rapido y verse bien.

  • Web de 1 a 4 secciones enfocada en conversion
  • Diseño responsive (mobile + desktop)
  • Estructura clara para presentar marca, servicio o producto
  • Botones directos a WhatsApp, Instagram o contacto
  • Formulario simple o CTA listo para recibir consultas

PLAN 02

Tienda basica para empezar a vender

Para marcas que ya quieren empezar a vender online, de forma simple y funcional.

  • Tienda base en WooCommerce (no Shopify)
  • Catalogo inicial cargado (productos y categorias)
  • Mercado Pago integrado + checkout funcionando
  • Paginas clave: inicio, tienda, producto, contacto y politicas
  • Guia breve para que puedas gestionar tu tienda
Quiero empezar a vender Mas info

PLAN 03

Tienda de marca con diseño personalizado

Para marcas donde no alcanza con "tener tienda": la experiencia visual, la navegacion y la identidad tambien tienen que vender.

  • Todo lo del Plan 2
  • Diseño alineado 100% a la identidad de marca
  • Home, catalogo y navegacion pensados para conversion
  • Fichas de producto mas cuidadas y persuasivas
  • Ideal para marcas donde lo visual impacta la venta
Quiero una tienda con identidad Mas info

PLAN 04

E-commerce avanzado para escalar

Para marcas que ya venden o quieren construir una base mas solida, medible y preparada para crecer con mas orden.

  • Arquitectura e-commerce avanzada segun tu operacion
  • Pagos, envios, variantes, cupones y reglas de venta
  • Metricas e integraciones (GA4, Pixel, WhatsApp, email)
  • Automatizaciones iniciales + salida a produccion
  • Base escalable para crecer sin rehacer todo

Mantenimiento mensual

Tu tienda no se entrega y desaparece.

Despues del lanzamiento, podes sumar un abono mensual para mantener tu tienda estable, actualizada y con soporte tecnico basico. La idea es que no tengas que ocuparte de la parte tecnica cada vez que surja algo.

  • Hosting y puesta online
  • Actualizaciones tecnicas y mantenimiento general
  • Backups de seguridad
  • Monitoreo basico y soporte tecnico
  • Pequeños ajustes o asistencia segun plan

El abono mensual se define segun el tipo de sitio, volumen de uso y nivel de soporte que necesites.
